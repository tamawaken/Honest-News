Corrections

Mistakes should be visible, not buried

The News Meta publishes a public corrections log so readers can see when a factual issue, source-mix issue, or process issue has been identified and fixed.

How corrections work

  • We review factual, editorial, and verification-related correction requests.
  • If a correction is confirmed, we update the affected page and log the change clearly.
  • Material changes describe what changed, why it changed, and when it changed.
  • We do not hide substantive fixes in silent edits.

How to report an issue

Send a clear description of the problem, the page URL, and the evidence supporting the claim. The more specific the report, the faster it can be checked.
