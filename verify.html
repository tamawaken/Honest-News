How Verification Works

Every analysis is signed with Ed25519 cryptography. Use this tool to verify that results haven't been tampered with.

Readiness note: Verification requires `keys/public_key.b64` and matching `*.sig` files for each article.

What This Proves

  • ✅ The analysis file matches the signature (no tampering)
  • ✅ The signature was created by this platform's private key
  • ✅ The results are authentic and unmodified

Note: This doesn't prove the analysis is "correct" — it proves the file you're viewing is what we generated and signed.
